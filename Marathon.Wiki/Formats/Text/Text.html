Text

The Text (MST) format for Sonic '06 defines the strings for in-game text.

After reading the BINA header, we end up at the first four important chunks of bytes.

00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00

  • This String is the signature of this BINA format.
  • This UInt32 is a pointer (plus 0x20) to the start of the null-terminated string table.
  • This UInt32 is the amount of text entries present in the file.
  • This UInt32 is a pointer (plus 0x20) to the internal name of the file.

After receiving this data, we can now start reading the offset table.

00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00

  • This UInt32 is a pointer (plus 0x20) to the name of this entry.
  • This UInt32 is a pointer (plus 0x20) to the UTF-16 encoded text for this entry.
  • This UInt32 is a pointer (plus 0x20) to the placeholder information for this entry.
  • Next entry in the offset table.
